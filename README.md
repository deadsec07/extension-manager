# Extension Manager — Quick Toggle for Chrome Extensions

Extension Manager is a Chrome management utility by A A Hasnat and HNE Technologies. It helps users enable and disable extensions in bulk, preserve safe states with snapshots, and work with focused include or exclude sets.

- Main site: https://hnetechnologies.com/
- Creator profile: https://deadsec07.github.io/

## Features
- Disable all except exclusions
- Enable only inclusions
- Snapshot and restore extension states
- Sync-compatible storage with no analytics
